Introduction

Markdown is a lightweight markup language for creating formatted text using a plain-text editor.

The syntax of Markdown consists of two parts, basic and extended.

Variants

Sites like GitHub, Bitbucket, Reddit, Diaspora, Stack Exchange, OpenStreetMap, and SourceForge use variants of Markdown to facilitate discussion between users.

Conclusion

Markdown is a lightweight markup language and easy to use.
But when using the extended syntax in your markdown, the rendering result varies depending on implementations.

So before using the extended syntax, you'd better refer to the documentation of the markdown rendering engine you are using.
