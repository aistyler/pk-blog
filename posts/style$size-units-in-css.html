Size units in CSS

peterhs.kang
December 10, 2021

Introduction

`px` `em` `rem` `vw` `vh` `vmin` `vmax`

What is all of these?

They are a unit used in CSS. But why such many units are necessary?

Let's figure out one by one.

No time? Then go to Conclusion

What is PX(PiXel)?

  • px is an absolute and fixed-size unit.
  • The physical size of 1px is dependent on a screen resolution.

What is EM(M)?

  • em is a relative unit to a parent element.
  • 1em is the 'font-size' defined in the parent element.
  • For example, 1em has a different pixel value by it's parent element.
div {
  font-size: 16px;
}
span {
  font-size: 8px;
}
p {
  font-size: 1em;
}

<div> <p> 1em is 16px here </p> </div>
<span> <p> 1em is 8px here </p> </span>

When to use EM?

  • It's good for responsive components.

What is REM(Root EM)?

  • rem is a relative unit to a root element.
  • 1rem is the pixel size defined in the root element.
  • For example, if you set the 'font-size' to 10px as follows, 1rem is translated into 10px by the browser.
html {
  font-size: 16px;
}
p {
  font-size: 1rem; /* <== translated into 10px */
}

When to use REM?

  • It's good for responsive web pages.
  • You can set the 'font-size' by divices using the 'media' query as follows:
html {
  /* Extra small devices (phones, 600px and down) */
  @media only screen and (max-width: 600px) {
    font-size: 10px;
  }
  /* Small devices (portrait tablets and large phones, 600px and up) */
  @media only screen and (min-width: 600px) {
    font-size: 12px;
  }

  /* Medium devices (landscape tablets, 768px and up) */
  @media only screen and (min-width: 768px) {
    font-size: 18px;
  }

  /* Large devices (laptops/desktops, 992px and up) */
  @media only screen and (min-width: 992px) {
    font-size: 26px;
  }

  /* Extra large devices (large laptops and desktops, 1200px and up) */
  @media only screen and (min-width: 1200px) {
    font-size: 34px;
  }
}

p {
  font-size: 1rem;
}

What is vh, vw, vmin and vmax?

  • v* are a relative unit to a viewport
  • 1vw is 1 percentage of a viewport width
  • 1vh is 1 percentage of a viewport height
  • 1vmin is min(1vh, 1vw)
  • 1vmax is max(1vh, 1vw)
  • In portrait orientation,
    • 1vmin is 1vw
    • 1vmax is 1vh
  • In landscape orientation,
    • 1vmin is 1vh
    • 1vmax is 1vw
